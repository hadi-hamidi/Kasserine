# 🍽️ مطعم القصرين - Al Kasserine Restaurant

موقع إلكتروني متكامل لمطعم القصرين مع لوحة تحكم إدارية.

## 🚀 المميزات

### 🌐 الموقع الرئيسي
- تصميم عصري ومتجاوب
- قائمة طعام ديناميكية
- نظام حجز الطاولات
- دردشة مباشرة
- معرض الصور

### 👨‍💼 لوحة التحكم الإدارية
- إدارة الأطباق والفئات
- رفع الصور أو استخدام الروابط
- إدارة الطلبات
- إدارة الحجوزات
- نظام دردشة مباشرة

## 🛠️ التقنيات المستخدمة

### Frontend
- React.js
- CSS3 مع تصميم متجاوب
- Socket.io للدردشة المباشرة

### Backend
- Node.js + Express.js
- MongoDB قاعدة البيانات
- JWT للمصادقة
- Multer لرفع الملفات
- Socket.io للدردشة المباشرة

## 📦 التثبيت والتشغيل

### 1. تثبيت المتطلبات
```bash
# Frontend
npm install

# Backend
cd server
npm install
```

### 2. إعداد قاعدة البيانات
```bash
# إنشاء ملف البيئة
cp server/env.example server/.env

# تعديل إعدادات قاعدة البيانات في .env
```

### 3. تشغيل المشروع
```bash
# تشغيل Backend
cd server
npm start

# تشغيل Frontend (في terminal جديد)
npm start
```

### 4. الوصول للمواقع
- الموقع الرئيسي: http://localhost:3000
- لوحة التحكم: http://localhost:5000/admin

## 📁 هيكل المشروع

```
├── src/                    # Frontend (React)
│   ├── components/         # مكونات React
│   └── App.js             # المكون الرئيسي
├── server/                # Backend (Node.js)
│   ├── admin/            # لوحة التحكم
│   ├── models/           # نماذج قاعدة البيانات
│   ├── routes/           # مسارات API
│   ├── scripts/          # سكريبتات الإعداد
│   └── uploads/          # الملفات المرفوعة
└── public/               # الملفات العامة
```

## 🎯 الاستخدام

1. **إضافة أطباق جديدة:** اذهب للوحة التحكم ← القائمة ← إضافة طبق جديد
2. **إدارة الفئات:** اذهب للوحة التحكم ← الأقسام
3. **متابعة الطلبات:** اذهب للوحة التحكم ← الطلبات
4. **إدارة الحجوزات:** اذهب للوحة التحكم ← الحجوزات
